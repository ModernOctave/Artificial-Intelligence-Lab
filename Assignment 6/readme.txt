# Installation
First install the scikit-learn package using pip:
    pip install scikit-learn

# Running
Run the program using the following command:
    python3 1.py spambase/spambase.data

# Usage
    python3 1.py <path_to_data> [C]

- path_to_data is the path to the data file.
- C is an optional argument to specify C (regularization constant) to use, if not specified 1 will be considered. (Larger values will require larger time to train!)
